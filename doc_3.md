# Reverse Traversal of a Linked List

## Data Structure
A singly linked list is used. Each node contains:
- An integer data value
- A pointer to the next node

## Functions Implemented
- `insert()` : Inserts a node at the beginning of the list.
- `display()` : Displays the linked list in forward order.
- `reverseTraversal()` : Prints the linked list in reverse order using recursion.

## main() Overview
The `main()` function:
1. Creates a linked list
2. Inserts sample values
3. Displays the list in forward order
4. Displays the list in reverse order

## Sample Output
Linked List (Forward): 40 -> 30 -> 20 -> 10 -> NULL
Linked List (Reverse): 10 -> 20 -> 30 -> 40 -> NULL
